# Damn Vulnerable Web Application (DVWA) – Security Testing & Mitigation Project

Repository ini berisi dokumentasi dan hasil pengerjaan Proyek Akhir Mata Kuliah Teknologi Keamanan Siber yang berfokus pada pengujian dan mitigasi kerentanan keamanan aplikasi web menggunakan Damn Vulnerable Web Application (DVWA) sebagai media simulasi.

Proyek ini dilakukan untuk memahami bagaimana celah keamanan pada aplikasi web dapat dieksploitasi, serta bagaimana penerapan mekanisme keamanan yang tepat mampu mencegah serangan tersebut. Seluruh pengujian dilakukan pada lingkungan lokal yang aman dan terkontrol.

**Ruang Lingkup Pengujian**
Beberapa jenis kerentanan keamanan web yang diuji dalam proyek ini meliputi:
• SQL Injection
• Command Injection
• Cross-Site Scripting (XSS)
• File Upload Vulnerability
• Cross-Site Request Forgery (CSRF)

Setiap kerentanan diuji pada tingkat keamanan rendah (low) untuk membuktikan adanya celah, kemudian dilakukan mitigasi melalui perbaikan kode seperti penggunaan prepared statement, validasi dan sanitasi input, encoding output, pembatasan tipe file, serta penerapan CSRF token. Setelah mitigasi diterapkan, dilakukan pengujian ulang untuk memastikan bahwa serangan tidak lagi berhasil.

Tujuan Proyek:
1. Mempelajari konsep dasar keamanan aplikasi web
2. Memahami cara kerja serangan terhadap aplikasi web
3. Menganalisis dampak dari masing-masing jenis serangan
4. Menerapkan teknik mitigasi keamanan yang sesuai
5. Membuktikan efektivitas mitigasi melalui pengujian ulang
